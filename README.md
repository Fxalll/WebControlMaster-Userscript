# 网页控制大师 (Web Control Master)

[![MIT License](https://img.shields.io/badge/License-MIT-green.svg)](https://opensource.org/licenses/MIT)
[![GreasyFork](https://img.shields.io/badge/GreasyFork-安装-blue)](https://greasyfork.org/zh-CN/scripts/496216)

一个功能强大、高度集成的浏览器用户脚本，为您提供全方位的网页浏览增强体验。它集成了图片控制、隐私保护、自动化操作和页面监控等多种实用工具，旨在成为您日常浏览的瑞士军刀。

## 🌟 主要特性

### 🖼️ 图片控制大师
- **一键显隐**：快速隐藏或显示网页上的所有图片，保护隐私或提升阅读专注度。
- **悬停/点击放大**：支持通过悬停、点击或鼠标中键放大图片，方便查看细节。
- **"阅兵"模式**：将页面所有图片以整齐的网格形式排列，便于集中浏览和比较。
- **图片钉图**：放大图片后，可将其"钉"在屏幕上任意位置，方便多图对比或作为参考。
- **可视化辅助**：可开启图片外框虚线，鼠标悬停时显示"隐/显"和"放"按钮，操作直观。

### 🔒 隐私与安全
- **标签页伪装**：伪装当前网页的标题和图标（Favicon），可以设置为预设或自定义内容，保护浏览隐私。
- **网页隐私锁**：
    - **多种上锁模式**：支持手动上锁、切换标签页（失焦）自动上锁、闲置超时自动上锁。
    - **PIN码解锁**：设置4位数字PIN码，为特定网站提供一层轻量级防护。
    - **安全提醒**：提供“忘记密码跳过”选项，并在跳过后持续提醒您重置密码。
- **页面局部遮罩**：通过鼠标拖拽在页面上绘制矩形遮罩，遮盖敏感信息（如价格、用户名等）。

### ⚡ 自动化与效率
- **自动点击器**：
    - **可视化流程编排**：通过添加“延时”、“点击元素”、“点击位置”、“输入文字”、“运行脚本”等步骤，构建自动化流程。
    - **智能等待**：可等待指定元素加载完成后再执行下一步，提高流程稳定性。
    - **循环执行**：支持设定循环次数或无限循环，适合需要重复操作的任务。
- **页面监控器**：
    - **元素监听**：监控页面上特定元素（通过CSS选择器或XPath定位）的文本变化、数值变化、出现或消失。
    - **条件触发**：当满足设定条件时，可执行弹窗提醒、桌面通知、刷新页面、运行自动点击器或执行自定义JS脚本。
- **快捷文本**：
    - **自定义快捷键**：为常用文本（如邮箱、地址、回复模板）绑定组合键（键盘或鼠标）。
    - **全局输入**：在任何输入框或可编辑区域按下快捷键，即可快速填充预设文本。
- **文字替换**：在网页上自动替换指定的文字内容，可用于修正术语、玩梗或过滤不喜欢的词汇。
- **页面可编辑**：一键开启 `contentEditable` 模式，让您可以像编辑文档一样直接修改网页上的文字。

### 📊 信息仪表盘
- **悬浮面板**：在屏幕边缘显示一个可拖动的信息面板。
- **丰富的统计信息**：可自定义显示时间、当前域名、图片隐藏数量、内存占用、页面加载时间、鼠标移动距离、点击次数、滚动距离、预估阅读字数等。
- **数据统计**：支持“本次会话”和“今日”两种统计范围，数据持久化存储。

## 🚀 安装

1.  **安装用户脚本管理器**：您需要先安装一个浏览器扩展，如 [Tampermonkey](https://www.tampermonkey.net/) 或 [Violentmonkey](https://violentmonkey.github.io/)。
2.  **安装脚本**：点击上方 **GreasyFork** 徽章或访问 [此链接](https://greasyfork.org/zh-CN/scripts/496216) 安装脚本。

## 🕹️ 使用指南

1.  **初始状态**：脚本默认会对页面所有符合条件的图片应用“隐藏”效果，并将其记录在仪表盘中。
2.  **显示/隐藏图片**：
    - **点击仪表盘**（绿灯状态）：点击主面板可在“图片隐藏”和“图片显示”状态间切换。
    - **悬停操作**：将鼠标悬停在单个图片上，会出现“显/隐”和“放”按钮，可单独控制。
3.  **打开主菜单**：将鼠标悬停在仪表盘上，即可唤出功能主菜单。
4.  **常用快捷键**：
    - `Alt + H`：显示或隐藏主仪表盘。

## ⚙️ 数据存储与作用域

脚本的大部分配置都支持**三级作用域**，方便您精细化管理：

-   **仅当前页**：配置仅对当前的URL路径生效。
-   **当前网站**：配置对当前域名下的所有页面生效。
-   **全局通用**：配置对所有网站生效。

当配置在不同作用域下冲突时，优先级为：**仅当前页 > 当前网站 > 全局通用**。
